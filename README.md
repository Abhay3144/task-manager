# task-manager

A task management application built with React, Express, and SQLite.

## Features

- User authentication
- Project management
- Task tracking
- Team collaboration
- Due date notifications

## Run Locally

1. Install dependencies:
   ```
   npm install
   ```

2. Generate Prisma client:
   ```
   npx prisma generate
   ```

3. Run the app:
   ```
   npm run dev
   ```

4. Open http://localhost:3000

## Tech Stack

- React 19
- Express
- SQLite with Prisma
- TypeScript
- Vite
